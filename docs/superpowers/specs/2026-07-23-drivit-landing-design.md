**Дата:** 2026-07-23
**Тип:** одностраничный лендинг (landing page), запуск под дедлайн ранней цены 27 июля 2026.

## Что это и зачем
Продающая посадочная страница для женского психологического ретрита **ДРАЙВИТ** (19–22 августа 2026, Вадул-луй-Водэ, берег Днестра). Цель: набор участниц на **14 мест** + поддержка ценового дедлайна. Трафик — из Instagram ведущих (мобильный).

## Хостинг / тех
- **GitHub Pages**, репозиторий `drivit-retreat`, ветка `main`.
- Поддомен: `drivit.smartspace.md` (CNAME → GitHub Pages, DNS на TopHost). До настройки DNS — работает на `<user>.github.io/drivit-retreat/`.
- Один файл `index.html`, self-contained (инлайн CSS/JS), **mobile-first**.
- Двуязычность **RO + RU**, переключатель, **RU по умолчанию** (весь исходный контент и аудитория русскоязычные).
- На время черновика — `<meta robots="noindex">` (не индексируем недоделку).

## Визуальная концепция — «Два голоса, одна женщина»
Напряжение между взрослой рефлексирующей женщиной (утончённый serif, тёплый закат, река, глубокий сумеречный плам) и её внутренним подростком (округлый геометрический дисплей, магента-драйв, блеск диско-шара). Закат над Днестром в час, когда зеркальный шар ловит последний свет.

- **Палитра:** paper #FBEDE7 (тёплый персиково-розовый) · ink/plum #2A0F30 · rose #E23C79 (акцент-драйв) · coral #FF7E5F (закат) · gold #F2C14E (блеск диско/премиум) · river #2E9E92 (спокойствие воды).
- **Типографика:** Spectral (serif — «голос женщины», эмоциональные заголовки) · Unbounded (округлый дисплей — «голос подростка», вордмарк/лейблы/цифры) · Golos Text (body). Все с кириллицей + Latin-ext (RO).
- **Тема:** светлая по умолчанию + тёмная (сумеречная) — обе проработаны через токены.
